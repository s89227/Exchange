CSS使用方式

  • 行內樣式(寫在HTML標籤中) 
    
<p style="color: green;">這是一個p標籤</p>
  • 內部樣式(寫在head中搭配selector) 
    
<head>
	<style>
		p{
			color: green;
		}
	</style>
</head>
<body>
	<p>這是一個p標籤</p>
</body>

CSS使用方式

  • 外部樣式(寫在.css檔中搭配selector) 
    
<head>
	≤link rel="stylesheet" type="text/css" href="setting.css">
</head>
<body>
	<p>這是一個p標籤</p>
</body>

Unit of Length

Unit of Length

  • px
  • cm
  • em
  • %
  • vh, vw

px, cm

  • 固定單位
  • 螢幕顯示用px
  • 輸出列印用cm

em, %, vh, vw

  • 比例單位
  • 預設字體大小為1em
  • 上層尺寸為100%
  • 視窗高度為100vh
  • 視窗寬度為100vw

calc函式


width: calc(100%/2-5px);
  • 計算長度
  • 固定單位和比例單位混用時很好用
