Part II:
Attribute

Attribute


<tagname attributename="value"></tagname>
<meta charset="utf-8">
<meter value="2" min="0" max="10">2 out of 10</meter>

<tagname attributename></tagname>
<input type="checkbox" name="vehicle" value="Car" checked>
  • 設定該元素額外屬性
  • 可以有多個屬性
  • 搭配JS控制

常用屬性

  • id 設定該元素的唯一名稱
  • class 類似id，但可以給多個元素使用
  • style 設定CSS樣式
  • hidden 隱藏元素

Part III:
Hyperlinks

超連結


<a href="http://s89227.github.io/Exchange/">超連結</a>
<a href="../../index.html">這也是超連結</a>
						

超連結
這也是超連結
  • a表示超連結
  • href為連結目標
  • 使用網址或相對路徑

路徑

  • 絕對路徑：第一排第二個
  • 相對路徑：我的右邊的前面
  • 外部網站使用絕對路徑(網址)
  • 內部網頁使用相對路徑

相對路徑

  • ./ 同層(可省略)
  • ../ 上一層
  • foldername/ 該資料夾下層

target屬性


<a href="../../index.html" target="_blank">超連結</a>

設定開啟連結的方式

value 效果
_blank 在新分頁或視窗開啟
_self 在這個頁面開啟(預設)
_parent 在母頁面開啟
_top 在這整個頁面開啟
framename 在指定的頁框開啟

download屬性


<a href="../../index.html" download>超連結</a>
<a href="../../index.html" download="home.html">超連結</a>
  • 設定download可下載連結目標
  • 可給值時預設下載的檔名
  • IE不支援

Part V:
Multimedia

圖片


<img src="pic.jpg" alt="A Image" width="304" height="228">
  • 單標籤
  • src設定路徑
  • alt設定替代文字
  • width, height設定寬高
  • 只設定一個會依比例縮放
  • 可用CSS代替

插圖


<figure>
	<img src="pic.png" alt="Picture">
	<figcaption>Fig1. - A Picture</figcaption>
</figure>
  • figure表示一組插圖
  • img設定圖片
  • figcaption設定圖片敘述

音訊


<audio src="music.mp3" controls autoplay loop>Not Support!</audio>

<audio controls>
	<source src="Wildlife1.mp4">
</audio>
  • 可接受格式：mp3, wav, ogg
  • src或source標籤設定音檔路徑
  • controls設定顯示控制面板
  • autoplay設定自動撥放
  • loop設定自動重複撥放
  • muted設定靜音
  • 標籤內文字設定不支援時顯示文字

視訊


<video src="video.mp4" controls muted poster="pic.png"></video>
  • 用法語audio類似
  • 可接受格式：mp4, webm, ogg
  • poster設定非播放時的縮圖

複數來源


<video controls poster="pic.png">
	<source src="source1.mp4">
	<source src="source2.mp4">
	<source src="source3.mp4">
	Not Support!
</video>
  • source標籤設定複數來源
  • src設定檔案路徑
  • 適用audio及video
  • 無法撥放時，瀏覽器依序切換來源
