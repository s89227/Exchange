HTML Introduction

2016/03/01

Part I:
網頁簡介

網頁

  • 一個檔案
  • 存放在某臺電腦中
  • 這台電腦必須連上網路
  • 透過網址取得這個檔案
  • 利用瀏覽器解讀內容後呈現

網頁的內容

  • 文字
  • 多媒體(圖片、音樂、影片等)
  • Applet(在頁面執行的副程式)
  • 超連結

網站

  • 許多網頁的集合
  • 存放在同一區域
  • 網站的起始頁面稱為首頁

網頁設計

大致分為前端、後端

  • 前端：UI、UX設計
  • 後端：程式設計
  • 全端：22K的產物

Part III:
HTML架構

基本架構


<!doctype html>
<html>
    <head>
        <title>我是頁面標題</title>
    </head>
    <body>
        內容
        <!--我是註解-->
    </body>
</html>

HTML Tag

用<和>夾著標籤名

網頁的基本元素

分成單標籤和雙標籤

單標籤<meta>, <br>, ...

雙標籤<p></p>, <title></title>, ...

動手做做看

  1. 打開編輯軟體
  2. 存檔為"檔名.html"
  3. 寫出HTML的基本架構
  4. 寫一些內容
  5. 再次存檔並用瀏覽器打開
  6. 咦？

檔頭


<head>
	<!--寫在這裡-->
</head>

整個頁面的大設定

  • 文字編碼
  • 標題
  • 資訊
  • 載入外部檔案

文字編碼


<meta charset="utf-8" >
  • 各語言都有自己的編碼
  • 錯的編碼解讀就會出現亂碼
  • 萬國碼可適用大多數語言

視窗標題


<title>視窗標題</title>
  • 雙標籤
  • 標題
  • 視窗或分頁標籤上的名稱
  • 不會顯示在內容中

網頁內容


<body>
	<!--寫在這裡-->
</body>

所有內容都寫在這邊

HTML5架構標籤


<body>
	<header>頁首
		<nav>導覽列</nav>
	</header>
	<main>
		內容，可分為若干個章節
		<section>章節</section>
		<section>章節</section>
		<section>章節</section>
	</main>
	<footer>頁尾</footer>
</body>

動手做做看

  1. 打開剛剛的網頁
  2. 在<head></head>中加上編碼設定
  3. 在<body></body>中加上HTML5結構標籤
  4. 存檔並用瀏覽器打開

Part IV:
常用HTML標籤

標題


<h1>最大</h1>
<h2>次大</h2>
<h3>第三大</h3>
<h4>第三小</h4>
<h5>次小</h5>
<h6>最小</h6>
  • 雙標籤
  • 由大到小
  • 不要拿來設定字的大小

段落


<p>
	段落內文
</p>
  • 雙標籤
  • 結束自動空一行(瀏覽器預設)
  • 不要拿來換行

換行


第一行<br>第二行
						

第一行
第二行
  • 單標籤
  • 不要拿來排版

分隔線


第一行
<hr>
第二行
						

第一行
第二行
  • 單標籤
  • HTML5 + CSS3 的時代來臨，逐漸被遺忘

動手做做看

  1. 打開剛剛的網頁
  2. 用剛剛學到的標籤幫內文做一些簡單的編排
  3. 存檔並用瀏覽器打開

強調標籤

  • 雙標籤
  • 可同時使用多組
  • 需一層包夾一層
  • 可以以CSS取代

強調標籤

效果 標籤

斜體 <em></em>
底線 <u></u>
刪除線 <del></del>

清單


<ul>
	<li>項目</li>
	<li>項目</li>
</ul>
  • 用於條列式
  • 清單標籤<ul></ul>包夾項目標籤<li></li>
  • 分成有編號<ol></ol>和無編號<ul></ul>
  • 清單中可以再有清單

Demo

表格


<table>
	<tr>
		<th>標題</th>
		<th>標題</th>
	</tr>
	<tr>
		<td>內容</td>
		<td>內容</td>
	</tr>
</table>
  • tr表示列
  • td表示欄
  • 展示大量有規則的資料
  • 不要用來排版
