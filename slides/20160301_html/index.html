Part I:
網頁簡介

網頁

  • 一個檔案
  • 存放在某臺電腦中
  • 這台電腦必須連上網路
  • 透過網址取得這個檔案
  • 利用瀏覽器解讀內容後呈現

網頁的內容

  • 文字
  • 多媒體(圖片、音樂、影片等)
  • Applet(在頁面執行的副程式)
  • 超連結

網站

  • 許多網頁的集合
  • 存放在同一區域
  • 網站的起始頁面稱為首頁

網頁設計

大致分為前端、後端

前端：UI、UX設計

後端：程式設計

Part III:
HTML架構

基本架構


<!doctype html>
<html>
    <head>
        <title>我是頁面標題</title>
    </head>
    <body>
        內容
        <!--我是註解-->
    </body>
</html>

HTML Tag

用<和>夾著標籤名

網頁的基本元素

分成單標籤和雙標籤

單標籤<meta>, <br>, ...

雙標籤<p></p>, <title></title>, ...

動手做做看

  1. 打開編輯軟體
  2. 存檔為"檔名.html"
  3. 寫出HTML的基本架構
  4. 寫一些內容
  5. 再次存檔並用瀏覽器打開
  6. 咦？

<head></head>

整個頁面的大設定

  • 語系
  • 標題
  • 資訊
  • 載入外部檔案

<title></title>

  • 雙標籤
  • 標題
  • 視窗或分頁標籤上的名稱
  • 不會顯示在內容中

<meta>

  • 單標籤
  • 設定或標記資訊

<meta charset="utf-8"><!--設定文字編碼-->
<meta name="keywords" content="web"><!--設定關鍵字-->
<meta name="author" content="Kei"><!--設定作者-->

<link>

  • 單標籤
  • 載入外部檔案

<link rel="stylesheet" href="css/general.css"><!--載入樣式檔-->
<link rel="icon" href="img/icon.ico" type="image/x-icon"><!--icon圖檔-->

動手做做看

  1. 打開剛剛的網頁
  2. 在<head></head>中加上meta設定
  3. 存檔並用瀏覽器打開

Part IV:
常用HTML標籤

標題

  • <h1></h1>~<h6></h6>
  • 雙標籤
  • 由大到小
  • 不要拿來設定字的大小

段落

  • <p></p>
  • 雙標籤
  • 結束自動空一行(瀏覽器預設)
  • 不要拿來換行

換行

  • <br>
  • 單標籤
  • 不要拿來排版

分隔線

  • <hr>
  • 單標籤
  • HTML5 + CSS3 的時代來臨，逐漸被遺忘

動手做做看

  1. 打開剛剛的網頁
  2. 用剛剛學到的標籤幫內文做一些簡單的編排
  3. 存檔並用瀏覽器打開
